# 🌦 Weather App - Proyecto Estrella

Aplicación de consola desarrollada en Python que permite consultar el clima en tiempo real utilizando una API externa, con almacenamiento local de datos, historial y sistema de favoritos.

---

## 🚀 Características

* 🌍 Consulta del clima en cualquier ciudad
* 🔗 Consumo de API real (OpenWeatherMap)
* 💾 Persistencia de datos con JSON
* 📜 Historial de búsquedas con información completa
* ⭐ Sistema de favoritos
* 🗑 Eliminación de registros
* 🕒 Registro de fecha y hora
* 🎨 Interfaz mejorada con emojis dinámicos
* 🔐 Uso de variables de entorno (.env)

---

## 🛠 Tecnologías utilizadas

* Python 3
* Requests
* JSON
* API REST

---

## 🧠 Aprendizajes clave

Este proyecto ha sido desarrollado para reforzar:

* Consumo de APIs en Python
* Manejo de estructuras de datos
* Persistencia de información
* Modularización del código
* Manejo de errores
* Buenas prácticas (uso de .env)
